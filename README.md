# ai-search
wrtn.ai의 ai 검색 기능

## 사용 예시

### 코드
```js
const { search } = require("./wrtn");

(async () => {
    const message = await search("송몽규라는 인물에 대해 알려줘"); // {success: ..., message: ...}
    console.log(message.message);
})();
```
